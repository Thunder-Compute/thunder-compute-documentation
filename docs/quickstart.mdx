## Getting Started

### 1. Install the CLI

```bash
pip install tnr
```

**Windows Users**: If you encounter permissions issues:
- Add the `--user` flag to the install command
- Add the Python user scripts directory to your PATH

### 2. Authentication

```bash
tnr login
```

After running this command, navigate to the console to [generate a token](https://console.thundercompute.com/settings/tokens).

### 3. Setup Billing

Visit the console to [add billing information](https://console.thundercompute.com/settings/billing) for your account.

### 4. Create Your First Instance

```bash
tnr create
```

### 5. Connect to Your Instance

```bash
tnr connect 0
```

That's it! You're now ready to use Thunder Compute.
