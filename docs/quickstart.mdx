1.  Install the Thunder Compute CLI
`
$ pip install tnr
`
2.  Log in and follow the instructions to navigate to the [console](https://console.thundercompute.com/login) and generate a token
`
$ tnr login
`
> Note: Windows users may need to add the `--user` flag `$ pip install --user tnr` and add the Python Scripts directory to their PATH
3.  In the console, add billing information to activate your account
4.  Create a new instance
`
$ tnr create
`
5.  Connect to this instance
`
$ tnr connect 0
`
