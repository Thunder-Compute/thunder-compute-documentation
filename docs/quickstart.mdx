## Getting Started

### Install the CLI

```bash
pip install tnr
```

<Note>
  **Windows Users**: If you encounter permissions issues, make sure the Python user scripts directory is added to your path.
</Note>

### Authenticate the CLI

```bash
tnr login
```

After running this command, navigate to the console to [generate a token](https://console.thundercompute.com/settings/tokens).

### Add a payment method

Visit the console to [add a payment method](https://console.thundercompute.com/settings/billing) to your account.

### Create Your First Instance

```bash
tnr create
```

To check on your instance

```bash
tnr status --wait
```

Once your instance is ready

```bash
tnr connect 0
```

That's it! You're now ready to use Thunder Compute.
