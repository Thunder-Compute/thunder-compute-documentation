1.  Run `$ pip install tnr` to install the Thunder Compute CLI
2.  Run `$ tnr login` and navigate to the to console to [generate a token](https://console.thundercompute.com/settings/tokens)
> Note: Windows users may need to add the `--user` flag `$ pip install --user tnr` and add the Python user scripts directory to their PATH
3.  In the console, [add billing information](https://console.thundercompute.com/settings/billing) to start using your account
4.  Create a new instance with `$ tnr create`
5.  Connect to this instance with `$ tnr connect 0`
