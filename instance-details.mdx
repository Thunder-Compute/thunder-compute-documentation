## Technical Specifications

- **CPU**: E series CPU instances in Azure
- **Networking**: 7Gbps Egress/Ingress
- **IP Address**: Dynamic
- **Location**: U.S. (region varies)

## Pre-installed Software

The latest CUDA drivers are pre-installed on every Thunder Compute instance. The following software is also available in the default environment. For more information about software compatibility, see our [Compatibility guide](/compatibility).

### CUDA

- **CUDA Version**: 12.9
- **CUDNN Version**: 9.0

### Core Libraries

- **PyTorch**: version 2.7.1 (CUDA-compatible)
- **JupyterLab**
- **Docker**: (see [Docker on Thunder Compute](/guides/using-docker-on-thundercompute))
- Other common libraries like `Numpy` and `Pandas` are also pre-installed.
